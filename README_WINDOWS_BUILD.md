# راهنمای ساخت فایل نصبی ویندوز (EXE)

این پروژه شامل یک Backend با **FastAPI (Python)** و یک Frontend با **React** است که برای تبدیل به یک فایل اجرایی تک‌فایل (`.exe`) در ویندوز آماده شده است.

## پیش‌نیازها
برای ساخت فایل نصبی، باید نرم‌افزارهای زیر روی سیستم شما نصب باشند:
1. **Python 3.8+** (ترجیحاً نسخه 64 بیتی)
2. **Node.js & npm** (برای بیلد گرفتن از React)

## مراحل ساخت فایل EXE

### روش خودکار (پیشنهادی)
کافیست روی فایل `build_windows.bat` دابل کلیک کنید. این اسکریپت تمام مراحل زیر را به صورت خودکار انجام می‌دهد:
1. نصب کتابخانه‌های پایتون (`requirements.txt`).
2. نصب پکیج‌های Node.js در پوشه `frontend`.
3. بیلد گرفتن از پروژه React (خروجی در `frontend/dist`).
4. بسته‌بندی نهایی با استفاده از PyInstaller.

### روش دستی
اگر ترجیح می‌دهید مراحل را دستی انجام دهید:

#### 1. نصب وابستگی‌های پایتون
```bash
pip install -r requirements.txt
```

#### 2. بیلد گرفتن از فرانت‌اند
وارد پوشه فرانت‌اند شوید و دستور زیر را اجرا کنید:
```bash
cd frontend
npm install
npm run build
cd ..
```
*نکته: مطمئن شوید که پوشه `frontend/dist` ساخته شده باشد.*

#### 3. ساخت فایل EXE
با استفاده از PyInstaller و فایل تنظیمات، دستور زیر را اجرا کنید:
```bash
pyinstaller --clean build_exe.spec
```

## خروجی نهایی
پس از اتمام موفقیت‌آمیز مراحل، فایل اجرایی `MyProfessionalApp.exe` در پوشه `dist` قرار خواهد داشت.
- این فایل شامل تمام کدهای پایتون و فایل‌های استاتیک React است.
- برای اجرای برنامه روی سیستم‌های دیگر، نیازی به نصب پایتون یا نود جی‌اس نیست.

## نکات مهم
- **آیکون برنامه:** اگر می‌خواهید برنامه آیکون اختصاصی داشته باشد، مسیر فایل `.ico` خود را در فایل `build_exe.spec` در بخش `icon=` قرار دهید.
- **حالت کنسول:** در حال حاضر پنجره سیاه رنگ کنسول مخفی است (`console=False`). اگر نیاز به دیدن لاگ‌ها دارید، در فایل spec مقدار آن را به `True` تغییر دهید.
- **حجم فایل:** به دلیل فشرده‌سازی (UPX)، حجم فایل نهایی کمتر خواهد بود، اما ممکن است آنتی‌ویروس‌ها به صورت کاذب به آن گیر دهند. در صورت نیاز می‌توانید `upx=True` را در فایل spec به `False` تغییر دهید.
