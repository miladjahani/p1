# Heap Master Pro - نرم‌افزار حرفه‌ای طراحی و مدیریت هیپ لیچینگ

## معرفی
Heap Master Pro یک نرم‌افزار تحت وب پیشرفته برای طراحی، شبیه‌سازی و مدیریت پدهای هیپ لیچینگ (به ویژه مس و طلا) است. این نرم‌افزار به صورت کامل روی GitHub Pages قابل اجرا بوده و از هیچ بک‌اند سنتی استفاده نمی‌کند.

## ویژگی‌های اصلی

### الف) ماژول محاسبات متالورژی و عملیاتی (Python-Powered)
- ✅ محاسبه دقیق حجم و تناژ با در نظر گرفتن توپوگرافی زمین
- ✅ مدل‌سازی OPEX (هزینه‌های عملیاتی شامل نیروی کار، سوخت دیزل، مصرف اسید)
- ✅ محاسبات محلول باردار (PLS) و کنترل اسیدیته (pH)
- ✅ زمان‌بندی تولید برای رسیدن به تارگت‌های کاتد مس (خلوص 99.99%)
- ✅ پیش‌بینی بازیابی مس بر اساس دبی، غلظت اسید و زمان (رگرسیون آماری)

### ب) موتور سه‌بعدی صنعتی (Three.js)
- ✅ نمایش دقیق لایه‌های زیرسازی (ژئوممبران، GCL)
- ✅ قابلیت اتصال پدها به صورت پیوسته (بدون درز)
- ✅ انیمیشن و شبیه‌سازی بصری جریان سیال
- ✅ ابزارهای اندازه‌گیری سه‌بعدی
- ✅ سیستم آبیاری (لاترال‌ها و امیترها)

### ج) داشبورد مدیریتی و شخصی‌سازی
- ✅ ذخیره‌سازی تنظیمات در localStorage
- ✅ Import/Export پروژه به صورت فایل JSON
- ✅ پنل تنظیمات پیشرفته ضرایب ثابت
- ✅ نمودارهای پیشرفت تولید و مصرف اسید (Chart.js)

## نحوه استقرار روی GitHub Pages

1. ریپازیتوری گیت‌هاب خود را ایجاد کنید
2. فایل‌ها را آپلود کنید
3. به Settings > Pages بروید
4. Branch را روی `main` و پوشه را روی `/ (root)` تنظیم کنید
5. پس از چند دقیقه، سایت شما در دسترس خواهد بود

## نحوه استفاده

### ایجاد پد جدید
1. از دکمه‌های "کنار هم (راست)"، "کنار هم (جلو)" یا "طبقه بالا" استفاده کنید
2. ابعاد و پارامترهای پد را در پنل سمت راست وارد کنید
3. تغییرات به صورت خودکار در نمای سه‌بعدی اعمال می‌شود

### مشاهده نمای سه‌بعدی
- روی دکمه شناور "نمای سه‌بعدی" کلیک کنید
- با موس می‌توانید صحنه را بچرخانید، زوم کنید و جابجا شوید
- از ابزارهای کنترل (بارش، وایرفریم، برچسب‌ها) استفاده کنید

### Export/Import پروژه
- از منوی تنظیمات، گزینه "Export Project" را انتخاب کنید
- فایل JSON دانلود شده را می‌توانید بعداً Import کنید

## محاسبات متالورژیکی

### فرمول بازیابی مس
```python
recovery = base_recovery +
           8.5 * log(flow_rate/50 + 1) +
           6.2 * log(acid_conc/10 + 1) +
           12.0 * sqrt(time_days/30) -
           0.15 * time_norm² +
           effect_of_grade +
           effect_of_particle_size
```

### محاسبات OPEX
- مصرف اسید: 5 کیلوگرم بر تن سنگ معدن
- مصرف سوخت: 0.3 لیتر دیزل بر تن
- هزینه سربار: 15% مجموع هزینه‌های مستقیم
